# AT-03
# Тестирование API (TheCatAPI)

Проект для практики написания тестов к API с использованием pytest и моков.

## Описание

Проект содержит:
1. Функцию `get_cats()` для получения данных с TheCatAPI
2. Тесты, проверяющие успешный и неуспешный сценарии работы функции

## Технологии

- Python 3.x
- pytest
- requests
- pytest-mock (для мокинга)

## Установка

1. Клонировать репозиторий
2. Установить зависимости:
   ```bash
   pip install pytest requests pytest-mock
Запуск тестов
bash
pytest -v
Что проверяют тесты
test_get_cats - проверяет успешный запрос к API и возврат корректных данных

test_get_cats_error - проверяет обработку ошибки (код 404)
